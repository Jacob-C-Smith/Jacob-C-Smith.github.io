G10 Web

G10 Web will be an implementation of the G10 game engine on the web browser. New technologies like WebGPU and WebAssembly are enabling high fidelity, high performance interactive media on ordinary webpages. I know next to nothing about web development, and that's why I started porting G10 to the browser. This page discusses the scope of this project, how it interacts with the rest of G10, and the reasoning behind these decisions.

Scope

G10 Web loads G10 assets. This includes instance configurations, shaders, AIs, and anything you might create with gport. G10 Web will implement the scheduler, and support a number of client tasks. You will be able to implement your own tasks if you want. The G10 Web API is written in JavaScript, and closely resembles the G10 Desktop API.
