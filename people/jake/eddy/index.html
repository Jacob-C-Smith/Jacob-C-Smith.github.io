eddy lang

Written by Jacob Smith

!!! eddy is still being developed !!!

The eddy language is a programming language I'm designing. I will henceforth be refering to "The eddy language" as "eddy" or "eddy lang" for brevity. The name comes from my dog, Eddy, whom I love more than life itself. Yes, the logo is a picture of Eddy.

eddy is a specialized programming language. That is to say, it is not a general purpose programming language. eddy was purpose-built to process data in large lists. eddy features 4 file types.

Extension Meaning Description
.eds EDY Source code Source code
.ede EDY Executable Intermediate between source and JIT compiled binary
.edd EDY Data An array of eddy values
.sced SChematic EDY Defines relations between a set of eddy programs
