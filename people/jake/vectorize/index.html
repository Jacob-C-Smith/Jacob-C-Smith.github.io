EDY lang

Written by Jacob Smith

!!! EDY is still being developed !!!

The EDY language is a programming language I'm designing. I will henceforth be refering to "The EDY language" as "EDY" or "EDY lang" for brevity. The name comes from my dog, Eddy, whom I love more than life itself. Yes, the logo is a picture of Eddy.

EDY is a specialized programming language. That is to say, it is not a general purpose programming language. EDY was purpose-built to process data in large lists. EDY features 4 file types.

Extension Meaning Description
.eds EDY Source code Source code
.ede EDY Executable Intermediate between source and JIT compiled binary
.edd EDY Data An array of EDY values
.sced SChematic EDY Defines relations between a set of EDY programs
